# Task Manager CLI

Простое консольное приложение для управления списком задач. Задачи сохраняются в JSON-файле, а управление ими осуществляется через набор команд.

## Возможности

- Добавление задач с описанием и сроком выполнения.
- Просмотр списка задач с фильтрацией по статусу и дате завершения.
- Обновление описания, названия и сроков, а также очистка полей.
- Отметка задач как выполненных и удаление.

## Требования

- Python 3.11+

Внешние зависимости не требуются — приложение использует только стандартную библиотеку.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

(Создание виртуального окружения необязательно, но рекомендуется.)

## Запуск тестов

```bash
python -m unittest discover -s tests
```

## Использование

```bash
python -m task_app.main --help
```

Основные команды:

- `add` — создать новую задачу.
- `list` — показать задачи (с фильтрами `--completed yes|no` и `--due-before YYYY-MM-DD`).
- `update` — изменить поля задачи.
- `complete` — отметить задачу выполненной (`--undo` для снятия отметки).
- `delete` — удалить задачу.

По умолчанию данные сохраняются в файл `tasks.json` в текущей директории. Можно указать путь через `--storage path/to/file.json`.
